💳 Loan Default Prediction (Credit Risk Modeling)
A Machine Learning project that predicts whether a customer is likely to default on a loan using Logistic Regression. The project analyzes customer financial and credit-related information to assess loan repayment risk.

🚀 Features
Data preprocessing and cleaning
Handling missing values
Categorical feature encoding
Exploratory Data Analysis (EDA)
Loan default prediction using Logistic Regression
Model evaluation using classification metrics
Data visualization using graphs and heatmaps

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

📂 Dataset Features
The dataset includes:
Person Age
Income
Home Ownership
Employment Length
Loan Intent
Loan Grade
Loan Amount
Interest Rate
Credit History Length
Previous Default History
Loan Status (Target Variable)

⚙️ Project Workflow
Load Dataset
Data Cleaning & Preprocessing
Handle Missing Values
Encode Categorical Columns
Perform Exploratory Data Analysis
Train Logistic Regression Model
Predict Loan Default
Evaluate Model Performance

▶️ Installation
Clone Repository
git clone <your-github-repo-link>cd <project-folder>

Install Dependencies
pip install pandas numpy matplotlib seaborn scikit-learn

▶️ Run Project
python main.py

📊 Output
The model predicts:

0 → Customer is NOT likely to default
1 → Customer is likely to default

The project also generates:
Accuracy Score
Confusion Matrix
Classification Report
Loan Default Distribution Graph
Correlation Heatmap

📈 Model Used
Logistic Regression
